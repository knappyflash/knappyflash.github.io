Instructions:
  • Type a Username.
  • Peer1: Click Create Offer.
  • Peer1: Wait for offer to be generated in the Local SDP textbox.
  • Peer1: Click Send Local SDP To Firebase.
  • Peer1: Click Read Offer From DB. (Your new offer should be there)
  • Peer2: Click Read Offer From DB. (Peer1's Offer should be there)
  • Peer2: Click Use Offer As Remote.
  • Peer2: Click Set Remote.
  • Peer2: Wait for answer to be generated in the Local SDP.
  • Peer2: If it does not generate an answer in Local SDP click Set Remote again. (Takes a while sometimes)
  • Peer2: Click Send Local SDP To Firebase.
  • Peer2: Click Read Answer From DB. (Your new Answer should be there)
  • Peer1: Click Read Answer From DB. (Peer2's new Answer should be there)
  • Peer1: Click Use Answer As Remote.
  • Peer1: Click Set Remote.
  • You both should now be connected and can talk to each other.
  • Accept any pop ups that may prompt for perms to use mic.
  • Once connected click Clear Voice Chat DB. (to remove the data from the database)
Other Instructions:
  • If you do not want to send data to the database, you have the option to use the Generate TXT File, and email it to your friend.
  • You and your friend can exchange your txt files for the handshake isntead of using the database.
