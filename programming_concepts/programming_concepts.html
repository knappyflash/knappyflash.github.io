Lesson

Get ready to code!

I think everyone should know how to read and write code, just like everyone should know how to read and write. You will not hear anyone say, reading and writing is not for everyone. Maybe back when the printing press was invented. You do hear people say, coding is not for everyone. I disagree. Reading and writing is obviously a super powerful tool in anyone’s life. SO IS CODING! It is what you do with it. I call computers anything machines. Some of the most “successful” people on earth are coders. They have benefited from the world’s coding illiteracy. Lately it seems like there has been a social push to decrease coding literacy. Not everyone needs to be Charles Dickens but knowing how to code is essential.
